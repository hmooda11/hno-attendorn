# HNO-Attendorn Praxis

Moderne statische Website fuer die HNO-Attendorn Praxis in Attendorn mit Wael Alyoussef als Facharzt fuer Hals-Nasen-Ohrenheilkunde.

## Lokale Entwicklung

```bash
npm install
npm run dev
```
